# Assistant Chat API
This project is a simplified version of OpenAIâ€™s Assistant API built using **FastAPI**, **SQLite**, and **Gemini API** (from Google AI). It allows you to:

- Create Assistants with a name and system instructions
- Start chats with those Assistants
- Send messages and receive LLM generated responses
- Persist all data across restarts



###  Tech Stack

- **Language**: Python 3.10+
- **Web Framework**: FastAPI
- **Database**: SQLite (with SQLAlchemy ORM)
- **LLM**: Gemini API
- **Testing**: pytest
